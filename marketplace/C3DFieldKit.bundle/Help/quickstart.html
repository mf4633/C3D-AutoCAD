C3D Field Kit — Quick Start

26 LISP commands for Civil 3D / AutoCAD. By Michael Flynn, PE.

After install

  1. Restart Civil 3D (or AutoCAD).
  2. Open the Plug-Ins tab → C3D Field Kit panel.
  3. Click a tool, or type a command at the prompt (e.g. LABELACRES).

No APPLOAD, Support Path, or Trusted Location setup is required.

Units

Macros assume US survey feet in drawing units. Area labels use square feet → acres (43560 sf/ac).
